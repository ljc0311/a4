# 抖音发布功能使用指南

## 📖 功能概述

抖音发布功能允许用户将生成的视频自动发布到抖音平台。该功能基于Selenium浏览器自动化技术，模拟用户操作完成视频上传、信息填写和发布过程。

## 🚀 使用前准备

### 1. 启动Chrome调试模式

在使用抖音发布功能前，必须先启动Chrome调试模式：

1. 在项目根目录找到 `start_chrome_debug.bat` 文件
2. 双击运行该文件
3. 等待Chrome浏览器自动启动
4. 确认看到"Chrome调试模式启动成功"的提示

> ⚠️ **重要提示**：保持Chrome调试模式窗口打开，关闭会导致发布功能无法使用

### 2. 登录抖音创作者平台

1. 在启动的Chrome浏览器中，访问 [抖音创作者平台](https://creator.douyin.com)
2. 使用手机扫码或其他方式登录您的抖音账号
3. 确认成功登录到创作者中心

### 3. 检查版本兼容性

如果遇到Chrome启动问题，请检查Chrome和ChromeDriver版本兼容性：

```bash
python scripts/check_chrome_compatibility.py
```

该脚本会自动检测Chrome版本并下载匹配的ChromeDriver。

## 🎬 发布视频步骤

1. 在程序主界面点击"一键发布"标签
2. 选择要发布的视频文件
3. 填写视频标题、描述和标签
4. 选择"浏览器自动化发布"选项
5. 勾选"抖音"平台
6. 点击"开始发布"按钮
7. 等待发布过程完成

## 🔧 故障排除

如果抖音发布功能无法正常工作，请按照以下步骤排查问题：

### 1. 运行诊断工具

```bash
python scripts/diagnose_douyin_publish.py
```

该工具会自动检查以下项目：
- Chrome安装状态
- ChromeDriver版本
- 调试端口状态
- Selenium依赖
- 网络连接
- 用户数据目录
- Chrome启动测试

### 2. 常见问题及解决方案

#### Chrome驱动启动超时

**症状**：日志中出现"Chrome驱动启动超时（30秒）"错误

**解决方案**：
1. 关闭所有Chrome浏览器窗口
2. 运行 `start_chrome_debug.bat` 重新启动调试模式
3. 确认Chrome调试端口已开启：`http://127.0.0.1:9222`

#### ChromeDriver版本不匹配

**症状**：Chrome启动失败，日志中提示版本不兼容

**解决方案**：
1. 运行 `python scripts/check_chrome_compatibility.py`
2. 按提示下载匹配的ChromeDriver版本

#### 无法连接到调试模式

**症状**：日志中出现"连接Chrome调试模式失败"

**解决方案**：
1. 确认已运行 `start_chrome_debug.bat`
2. 检查端口9222是否被其他程序占用
3. 重启计算机后重试

#### 抖音登录状态失效

**症状**：发布过程中提示需要登录

**解决方案**：
1. 在Chrome调试模式浏览器中重新登录抖音创作者平台
2. 确认登录状态后重试发布

## 📋 注意事项

1. **保持登录状态**：抖音平台的登录状态会定期失效，需要重新登录
2. **视频格式要求**：抖音推荐使用MP4格式，分辨率1080x1920，时长15-60秒
3. **内容合规**：确保视频内容符合抖音平台规范，避免违规内容
4. **发布频率**：避免短时间内频繁发布，可能触发平台风控
5. **调试模式**：每次使用发布功能前，确保Chrome调试模式已启动
