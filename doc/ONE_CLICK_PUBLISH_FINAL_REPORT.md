## 🚀 使用指南

### 1. 环境准备
```bash
# 1. 确保Chrome浏览器已安装
# 2. 下载对应版本的ChromeDriver
# 3. 启动Chrome调试模式（可选）
chrome.exe --remote-debugging-port=9222
```

### 2. 功能使用流程
1. **项目准备**: 在主界面加载包含视频的项目
2. **AI优化**: 点击"🎯 AI优化内容"自动生成发布内容
3. **平台选择**: 勾选要发布的平台（抖音、快手、小红书等）
4. **参数调整**: 根据需要调整标题、描述、标签等
5. **一键发布**: 点击"🚀 开始发布"执行多平台发布
6. **状态监控**: 实时查看发布进度和结果

### 3. 发布流程图
```
项目加载 → AI内容优化 → 平台选择 → 视频转换 → 
浏览器启动 → 登录检查 → 视频上传 → 内容填写 → 
发布提交 → 结果验证 → 状态更新 → 完成反馈
```
