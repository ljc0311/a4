## 🚀 使用方法

### 1. 环境准备
```bash
# 1. 安装Chrome浏览器
# 2. 下载对应版本的ChromeDriver
# 3. 启动Chrome调试模式
chrome.exe --remote-debugging-port=9222
```

### 2. 功能使用
1. **加载项目**: 在主界面加载包含视频的项目
2. **AI优化**: 点击"🎯 AI优化内容"自动生成标题、描述、标签
3. **选择平台**: 勾选要发布的平台（抖音、快手、小红书等）
4. **一键发布**: 点击"🚀 开始发布"执行多平台发布

### 3. 发布流程
```
项目加载 → AI内容优化 → 平台选择 → 视频上传 → 内容填写 → 发布提交 → 结果反馈
```
