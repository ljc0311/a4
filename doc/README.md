## 主要功能模块

### 五阶段分镜系统
- 文本创作和优化
- 语音生成（多TTS引擎）
- 图像生成（多AI引擎）
- 视频生成（多服务支持）
- 一键发布（多平台）

### 一键发布功能
支持以下平台：
- 抖音
- 快手
- 小红书
- B站
- 微信视频号
- YouTube Shorts

### 配置要求
- Chrome浏览器调试模式
- 相应平台的API密钥
- 网络代理配置（如需要）
