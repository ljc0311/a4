# ==================== 注意事项 ====================
# 1. 核心依赖：PyQt5 (GUI)、requests (网络)、aiohttp (异步)、Pillow (图像)
# 2. 一键发布功能需要 selenium (浏览器自动化)
# 3. 中文文本处理需要 jieba
# 4. 敏感数据加密需要 cryptography
# 5. 系统监控需要 psutil
# 6. 大部分AI功能通过API调用实现，无需本地AI库
# 7. 可选依赖根据具体功能模块按需安装
# 8. 如遇到缺失模块错误，请根据错误信息安装对应依赖
